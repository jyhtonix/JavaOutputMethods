Java Output Methods

Introduction to Programming - Visual Explanation for Diploma Students
1

System.out.println()

System.out.println("Hello, World!");

What it does: Prints text to the console and moves to the next line.

How to remember: Think of "println" as "print line". It prints and then goes to a new line.

Key characteristic: After printing, the cursor moves to the beginning of the next line.

Simple Analogy: Imagine you're writing on a piece of paper. With println, after writing something, you move your pen to the next line automatically.
2

System.out.print()

System.out.print("Hello, ");
System.out.print("World!");

What it does: Prints text to the console but stays on the same line.

How to remember: Think of "print" as just "print" without the "line" part.

Key characteristic: After printing, the cursor stays at the end of what was printed, ready for more text on the same line.

Simple Analogy: Imagine you're writing on a piece of paper. With print, you write something and keep your pen right where you finished, ready to write more on the same line.
3

System.out.printf()

int sum = 25;
System.out.printf("Sum is %d\n", sum);

What it does: Prints formatted text to the console. You can insert variables into specific places in the text.

How to remember: Think of "printf" as "print formatted".

Key characteristic: Uses placeholders like %d (for integers), %f (for decimals), %s (for text) to insert values into the output string.

Special note: \n at the end moves to the next line (like println does).

Simple Analogy: Imagine you have a form letter with blanks to fill in. With printf, you create a template with blanks (like %d), and then you provide the values to fill those blanks.
+

Import Scanner for Input

import java.util.Scanner;

What it does: This line tells Java you want to use the Scanner class, which is a tool for reading user input from the keyboard.

How to remember: Think of it like telling Java, "I need to use the Scanner tool from the java.util toolbox."

Key characteristic: You need to put this line at the very top of your Java file (before the class definition) to use Scanner.

Simple Analogy: Before you can use a special tool (like a calculator), you need to take it out of your toolbox. import java.util.Scanner; is like taking the Scanner tool out of Java's toolbox so you can use it.
